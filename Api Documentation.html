Gym Management API Documentation

Authentication

This API uses JWT authentication. To obtain a token, use:

  • Login: POST /api/token/

  • Refresh Token: POST /api/token/refresh/

Include the token in the Authorization header as Bearer <token> in requests requiring authentication.

User API

1. User Login

Endpoint: POST /api/user/

Request Body:

{
"username": "admin",
"password": "password123"
}

Response:

{
"id": 1,
"username": "admin",
"email": "admin@example.com",
"gym_name": "FitGym",
"phone_number": "1234567890"
}

2. Get User Details

Endpoint: GET /api/user/?id=<user_id>

Response:

{
"id": 1,
"username": "admin",
"email": "admin@example.com",
"gym_name": "FitGym",
"phone_number": "1234567890"
}

3. Update User Details

Endpoint: PATCH /api/user/?id=<user_id>

Request Body (Example):

{
"phone_number": "9876543210"
}

Gym User API

1. Get Gym Users

Endpoint: GET /api/gym-user/?gym-id=<gym_id>

Response:

[
{
"id": 1,
"name": "John",
"email": "john@example.com",
"phone_number": "1234567890"
}
]

2. Get Gym User by ID

Endpoint: GET /api/gym-user/?user-id=<user_id>

3. Create Gym User

Endpoint: POST /api/gym-user/?create=true

Request Body:

{
"name": "John",
"family": "Doe",
"email": "john@example.com",
"phone_number": "1234567890"
}

4. Update Gym User

Endpoint: PATCH /api/gym-user/?user-id=<user_id>

5. Delete Gym User

Endpoint: DELETE /api/gym-user/?user-id=<user_id>

Gym User Payment API

1. Get Payments

Endpoint: GET /api/payment/?gym-id=<gym_id> OR GET /api/payment/?user-id=<user_id>

2. Get Payment by ID

Endpoint: GET /api/payment/?payment-id=<payment_id>

3. Create Payment

Endpoint: POST /api/payment/?create=true

Request Body:

{
"user_id": 1,
"payed_amount": "100",
"subscription_duration": "1_months",
"payment_method": "cash"
}

4. Update Payment

Endpoint: PATCH /api/payment/?payment-id=<payment_id>

5. Delete Payment

Endpoint: DELETE /api/payment/?payment-id=<payment_id>

Logs API

1. Get Logs

Endpoint: GET /api/logs/?gym-user-id=<gym_user_id> OR GET /api/logs/?gym-id=<gym_id>

2. Create Log

Endpoint: POST /api/logs/

Request Body:

{
"gym_user": 1
}

3. Update Log

Endpoint: PATCH /api/logs/?log-id=<log_id>

Request Body:

{
"logout_time": "18:00:00"
}
